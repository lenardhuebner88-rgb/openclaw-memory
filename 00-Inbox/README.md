# Inbox Triage

## Zweck
- Alles Neue, Unklare oder noch nicht einsortierte landet zuerst hier.
- Nicht beim Erfassen nachdenken, erst später kurz sortieren.

## Triage-Regel
1. Gehört es zu einem laufenden Projekt? → `02-Projects/...`
2. Ist es dauerhafter Shared-Kontext? → `03-Agents/Shared/...`
3. Ist es operativer Systemzustand? → `03-Agents/OpenClaw/...` oder `04-Operations/...`
4. Ist es nur temporär oder unklar? → erstmal hier lassen
5. Ist es fertig und nur historisch? → `06-Archive/...`

## Nicht tun
- keine Root-Dateien neu erzeugen
- keine spontanen Sonderordner anlegen
- keine Todo-Sammlungen hier wachsen lassen

## Ziel
- Inbox klein halten
- schnell entscheiden
- Rest des Vaults sauber halten
